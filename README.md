
# python-inventory

This is a inventory project using python,
It has functions that can manipulate inventory data and logging the actions

## Run Locally

Change the current directory into python-inventory-keta-yusa
and run this command 
`Python kenta_coursework1.py`

## Features implemented
[x] Add item  
[x] View  inventory  
[x] Update Item  
[x] Remove Item  
[x] Exit  
[x] Load items from a text file  
[x] Save items to text file

## Any limitations
- If the user enter wrong category name twice, This app will continue to work
- If the user enter wrong item name twice, This app will cancel the procedure
